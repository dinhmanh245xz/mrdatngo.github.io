Các câu lệnh git cơ bản

  • Kiểm tra version git: git --version
  • Kiểm tra trạng thái: git status
  • Add 1 file vào bộ nhớ tạm: git add [filename]
  • Add tất cả thay đổi vào bộ nhớ tạm: git add .
  • Lưu thay đổi trong bộ nhớ tạm thành 1 log (commit): git commit -m "[message]"
  • Kiểm tra các lịch sử commit: git log
  • Update thay đồi từ local lên server: git push origin master

Các bước hoàn thành thay đổi từ local lên github

  1. git add .
  2. git commit -m "[message]"
  3. git push origin master
